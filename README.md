imgur-dl
========

Download albums from imgur.

```
usage: imgur-dl.py [-h] [-u USERNAME] [-a ALBUM_ID] [-s START_AT] client-id

Download albums from imgur.

positional arguments:
  client-id             imgur api client id

optional arguments:
  -h, --help            show this help message and exit
  -u USERNAME, --username USERNAME
                        user account to download (public albums)
  -a ALBUM_ID, --album-id ALBUM_ID
                        album id to download
  -s START_AT, --start-at START_AT
                        album/image to start at (resume)
```
